Hướng dẫn sử dụng

        

        

          

            

              
1

              

                
Bootstrap Admin

                

                  Nhập Admin Email và Master Password, bấm
                  Bootstrap Account để tạo tài khoản quản trị đầu tiên.
                

              

            

            

              
2

              

                
Đăng nhập

                

                  Dùng email + mật khẩu vừa tạo, bấm Login.
                  Nếu thành công, trạng thái token sẽ chuyển sang active.
                

              

            

            

              
3

              

                
Tạo Provider

                

                  Chọn Key (openai/anthropic/google/perplexity),
                  nhập API Base URL và Chat Path, rồi bấm
                  Create Provider.
                

              

            

            

              
4

              

                
Tạo Account

                

                  Dán Provider ID, nhập Token và Expiry
                  (nếu có), bấm Create Account.
                

              

            

            

              
5

              

                
Test Proxy

                

                  Chọn Model, nhập nội dung, bấm Call /proxy/chat.
                  Nếu bật Async, kiểm tra trạng thái job ở dưới.
                

              

            

          

          

            
Mẹo nhanh

            
    
              
  • Provider ID lấy bằng nút Load Providers.
    • 
              
  • Token chỉ hiển thị last4, full token cần admin.
    • 
              
  • Thử Async khi gọi model nặng để tránh timeout.
    • 
              
  • Hãy kiểm tra Health nếu proxy lỗi liên tục.
